# Features Information
- Tile Shortcut to run FreePPS script
- Notification Post for FreePPS Status
- Toast FreePPS Status (on Clicking Notification Post and on Execution of Tile)
- Update Tile State and Notification Post on Reboot

# Requirements
- [FreePPS Module](https://github.com/Seyud/FreePPS)
- Notification Permission
- Tile Adoption
- Root
