# 功能特性
- 磁贴运行 FreePPS 脚本
- 通知推送 FreePPS 状态
- Toast 显示 FreePPS 状态（通知按钮和点击磁贴时）
- 重启后更新磁贴状态和通知推送

# 要求
- [FreePPS 模块](https://github.com/Seyud/FreePPS)
- 通知权限
- Root 权限
